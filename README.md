# eml

**Language:** [English](README.en.md) · [简体中文](README.zh.md)

Archived experimental toy — negative results, not maintained.

期中考试无聊 + 额度没用完随便整的玩具 · 归档 · 负结果 · 不再维护

**inspired by**
Andrzej Odrzywołek, "All elementary functions from a single binary operator" (2026).

The idea of representing mathematical operations using a compact set of exp/log-based primitives motivated this project.

**Scope note:** full **256-node sweep** and **multi-node stacking** were planned but **not runnable locally**; repo ships **16-node single-tree** sweep only. Several numerics ideas (`log1p`/`expm1`, logsum **T** merge) were only partly sketched — see language READMEs § Abandoned ideas.
